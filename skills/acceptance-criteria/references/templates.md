# Acceptance Criteria Templates

## Template: Given/When/Then

```gherkin
Scenario: [name]
Given [context]
When [action]
Then [outcome]
And [additional outcome]
```

## Template: Rule Checklist

```markdown
Acceptance Criteria (Must Have):
- [ ] [measurable outcome]
- [ ] [measurable outcome]

Negative/Edge Scenarios:
- [ ] [failure handling]
- [ ] [boundary condition]

Out of Scope:
- [item]
```
