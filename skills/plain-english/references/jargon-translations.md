# Jargon Translations

Use these mappings to keep technical meaning while improving readability.

| Technical Term | Plain English |
| --- | --- |
| API | connection point between systems |
| Database | central data storage |
| Microservice | small independent application component |
| Container | packaged runtime unit |
| Latency | response delay |
| Throughput | processing capacity over time |
| Refactor | improve internal structure without changing outcome |
| Technical debt | deferred maintenance work |
| Incident response | organized process to handle security/service events |
| Dependency upgrade | update a supporting software component |
